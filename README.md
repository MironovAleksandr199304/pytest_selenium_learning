# pytest_selenium_learning

Учебный проект со стартовой структурой для изучения связки **pytest + Selenium**. Репозиторий содержит примеры настроек, фикстуры и простой тест, который работает с локальной HTML-страницей, чтобы можно было пробовать сценарии без доступа к интернету.

## Содержимое

- `requirements.txt` — основные зависимости (pytest, selenium, webdriver-manager).
- `pytest.ini` — регистрация пользовательских маркеров.
- `tests/conftest.py` — глобальные фикстуры и CLI-опция `--run-browser`.
- `tests/resources/login_page.html` — локальная страница для экспериментов.
- `tests/functional/test_login_form.py` — пример теста, использующего Selenium.

## Подготовка окружения

1. Установите Python 3.10+.
2. Создайте виртуальное окружение и активируйте его.
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## Запуск тестов

- Базовый запуск (все тесты, браузерные будут пропущены без флага):
  ```bash
  pytest -q
  ```

- Запуск тестов с браузером (головной режим, локальный HTML):
  ```bash
  pytest -q --run-browser
  ```

## Полезные идеи для самообучения

- Добавляйте новые страницы в `tests/resources/` и создавайте под них отдельные тесты.
- Экспериментируйте с ожиданиями (`WebDriverWait`) и разными локаторами.
- Настройте отчёты (например, `pytest-html`) и параметризацию сценариев.
- Интегрируйте линтеры/форматтеры (ruff/black) и CI для автоматической проверки.
